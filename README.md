# Budget vs Forecast Analysis Dashboard

## Overview
This Power BI dashboard provides a comprehensive financial analysis comparing **Budget vs Forecast** performance across regions, countries, IT departments, and cost elements. The report enables stakeholders to monitor financial trends, identify variances, and gain actionable insights for improved planning and decision-making.

---

## Key Features
- Budget vs Forecast variance analysis
- Financial KPI monitoring
- Regional and country-level performance tracking
- IT department and cost element analysis
- Interactive decomposition tree analysis
- Geographic performance visualization
- Dynamic slicers and drill-through functionality
- Trend analysis using interactive charts

---

## Dashboard Pages

### 1. Executive Overview
- High-level KPI summary
- Budget vs Forecast trend comparison
- Regional financial performance overview
- Variance percentage indicators
- Interactive filtering options

### 2. Regional Analysis
- Country-wise and regional breakdown
- Variance distribution analysis
- Geographic visualization using map charts
- Detailed financial comparison tables
- Scatter plot analysis for performance insights

### 3. Decomposition Analysis
Interactive decomposition tree allowing users to drill down into:
- IT Department
- Cost Element
- Country
- Cost Element Group

This helps identify key drivers behind financial variances.

---

## Business Problem
Organizations often struggle to monitor how forecasted spending aligns with planned budgets across multiple business units and regions. This dashboard helps finance and operational teams quickly identify:
- Overspending areas
- Forecast deviations
- Regional performance gaps
- Operational cost drivers
- Areas requiring financial optimization
